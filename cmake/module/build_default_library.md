# Выбор типа сборки

Этот CMake скрипт управляет выбором типа сборки проекта на основе значения опции `AE_OPTION_BUILD_TYPE_SHARED`. В
зависимости от её значения проект будет собран как динамическая или статическая библиотека.

## Описание

1. **Конфигурация типа сборки**:
    - Скрипт проверяет значение параметра `AE_OPTION_BUILD_TYPE_SHARED`.
    - Если параметр установлен в `ON`, проект будет собран как динамическая библиотека.
    - Если параметр установлен в `OFF`, проект будет собран как статическая библиотека.

2. **Логика сборки**:
    - Если `AE_OPTION_BUILD_TYPE_SHARED` равен `ON`, используется флаг `SHARED` для создания динамической библиотеки.
    - Если параметр равен `OFF`, используется флаг `STATIC` для создания статической библиотеки.

## Пример использования

```cmake
if (AE_OPTION_BUILD_TYPE_SHARED)
    # Создаём динамическую библиотеку
    add_library(${PROJECT_NAME} SHARED ${AE_TARGET_SOURCE_FILES})
else ()
    # Создаём статическую библиотеку
    add_library(${PROJECT_NAME} STATIC ${AE_TARGET_SOURCE_FILES})
endif ()
```

## Заключение

Этот CMake скрипт позволяет гибко настраивать тип сборки проекта, выбирая между динамической и статической библиотекой в
зависимости от пользовательских настроек.
