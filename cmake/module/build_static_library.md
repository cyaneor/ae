# Создание статической библиотеки

Этот CMake скрипт нацелен на создание статической библиотеки из исходных файлов проекта. Он выполняет несколько ключевых
задач, описанных ниже.

## Описание

1. **Создание статической библиотеки**:
   Скрипт создает статическую библиотеку с именем, основанным на названии проекта, добавляя суффикс "_static". Для
   сборки библиотеки используются исходные файлы, указанные в переменной `${AE_TARGET_SOURCE_FILES}`.

2. **Возможность использования в других программах**:
   Созданная статическая библиотека может быть использована для линковки с другими частями проекта или внешними
   программами, обеспечивая повторное использование и удобство в разработке.

3. **Преимущества использования статических библиотек**:
    - Упрощает сборку: все необходимые исходные файлы объединяются в один файл, что упрощает процесс компиляции и
      линковки.
    - Независимость от внешних библиотек: статическая библиотека встраивается в конечный исполнимый файл, что исключает
      необходимость в установке сторонних зависимостей.

## Пример команды для создания библиотеки

```cmake
add_library(${PROJECT_NAME}_static STATIC ${AE_TARGET_SOURCE_FILES})
```

Эта CMake команда создает статическую библиотеку с уникальным именем, основанным на названии проекта.

## Заключение

Этот CMake скрипт упрощает процесс создания и использования статических библиотек, обеспечивая удобство при разработке и
поддержке проектов, использующих общие компоненты.
