# Конфигурация CMake для библиотеки

Этот CMake файл задает различные опции конфигурации для сборки и управления параметрами библиотеки. Опции влияют на
поведение библиотеки, включая создание различных типов библиотек, экспорт и импорт символов, использование
многопоточности и управление памятью.

## Опции конфигурации

### `AE_OPTION_BUILD_TYPE_SHARED`

- **Описание**: Управляет сборкой проекта как общей (динамической) библиотеки.
- **Использование**:
    - `ON`: Сборка как общая библиотека.
    - `OFF`: Отключает сборку как общую библиотеку, проект будет собран как статическая.

### `AE_OPTION_BUILD_IMPORT`

- **Описание**: Определяет, должны ли символы быть импортированы при сборке общей библиотеки.
- **Использование**:
    - `ON`: Импорт символов включен.
    - `OFF`: Импорт символов отключен.

### `AE_OPTION_BUILD_EXPORT`

- **Описание**: Определяет, должны ли символы экспортироваться при сборке общей библиотеки.
- **Использование**:
    - `ON`: Экспорт символов включен.
    - `OFF`: Экспорт символов отключен.

### `AE_OPTION_SEGMENTED_ARCH`

- **Описание**: Указывает, используется ли целевая архитектура с сегментированной моделью памяти.
- **Использование**:
    - `ON`: Требуется использование сегментированной модели памяти.
    - `OFF`: Целевая архитектура не использует сегментированную модель памяти.

### `AE_OPTION_RUNTIME_ALLOCATOR_INIT_STDLIB`

- **Описание**: Определяет использование стандартной библиотеки для инициализации аллокатора в режиме выполнения.
- **Использование**:
    - `ON`: Используется стандартная библиотека для инициализации.
    - `OFF`: Не используется стандартная библиотека.

### `AE_OPTION_FILL_ZERO_AFTER_MEMORY_ALLOCATE`

- **Описание**: Указывает, должна ли недавно выделенная память инициализироваться нулями.
- **Использование**:
    - `ON`: Память заполняется нулями.
    - `OFF`: Память не инициализируется.

### `AE_OPTION_THREAD_LOCAL_VARIABLES`

- **Описание**: Указывает, должны ли статические переменные быть объявлены как `thread_local` для многопоточной
  безопасности.
- **Использование**:
    - `ON`: Статические переменные имеют модификатор `thread_local`.
    - `OFF`: Используется стандартное поведение для статических переменных.
